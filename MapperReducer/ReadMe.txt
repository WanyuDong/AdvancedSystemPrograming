1.About files
Three source files: mapper.c reducer.c combiner.c
Makefile: to create compiled code, mapper.o reducer.o combiner.o
run.sh: to show how to run and test the code.
test data: 
    input.txt: used to test mapper and combiner
    Mapper_Output.txt: used to test reducer

2.How to use
First, run make
Second, run ./run.sh to test combiner by useing input.txt
If you want to test mapper and reducer, just open run.sh, then comment and uncomment lines you wanted
